# TikTok & Amazon 服装数据系统使用指南

## 系统概述

这是一个自动化的服装竞品研究系统，帮助您收集、分析和监控TikTok和Amazon平台上的服装产品数据。系统提供了数据抓取、数据可视化、竞品分析等核心功能。

### 主要功能
- **数据抓取**：自动从TikTok和Amazon平台抓取服装产品信息
- **数据可视化**：通过Web仪表板查看实时数据和分析结果
- **竞品分析**：比较不同平台的产品价格、销量、评价等指标
- **数据导出**：支持多种格式的数据导出功能
- **监控告警**：实时监控系统运行状态和数据质量

### 系统组件
1. **数据抓取引擎**：核心的后端数据收集模块
2. **Web仪表板**：React构建的前端数据可视化界面
3. **数据库**：SQLite数据库存储产品信息
4. **配置文件**：灵活的YAML配置管理系统

## 支持的数据类型

### 产品信息
- **基本信息**：产品名称、品牌、价格、SKU
- **分类信息**：产品类别、标签、属性
- **销售数据**：销量排名、评价数量、评分
- **库存状态**：是否有货、库存数量
- **链接信息**：产品页面URL、图片链接

### 市场分析数据
- **价格趋势**：历史价格变化
- **销量分析**：热门产品排行
- **竞争分析**：同类产品价格对比
- **用户评价**：客户反馈分析

## 技术规格

### 系统要求
- **操作系统**：Windows 10/11, macOS 10.15+, Ubuntu 18.04+
- **Python版本**：3.8+
- **内存要求**：至少4GB RAM
- **存储空间**：至少2GB可用空间
- **网络**：稳定的互联网连接

### 依赖组件
- Python 3.8+
- SQLite数据库
- Node.js 16+ (前端仪表板)
- 支持的浏览器：Chrome 90+, Firefox 88+, Safari 14+
