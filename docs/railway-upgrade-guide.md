# Railway 升级到 Starter 计划指南

## 升级步骤

### 1. 登录 Railway 账户
- 访问 https://railway.app/
- 点击 "Login" 使用GitHub账户登录

### 2. 进入账户设置
- 登录后，点击右上角的账户头像
- 选择 "Settings" 或 "账户设置"

### 3. 查看当前计划
- 在设置页面找到 "Billing" 或 "订阅" 区域
- 查看当前的计划状态（应该是 "Free" 免费计划）

### 4. 升级到 Starter 计划
- 点击 "Upgrade to Starter" 或 "升级" 按钮
- 选择 **Starter** 计划（$5/月）
- 点击 "Subscribe" 或 "订阅"

### 5. 填写支付信息
- Railway 使用 Stripe 作为支付处理器
- 需要提供信用卡信息（Visa, MasterCard等）
- 输入账单地址信息

### 6. 完成支付
- 确认订单详情
- 点击 "Complete Payment" 或 "完成支付"
- 等待处理完成

### 7. 验证升级成功
- 返回仪表板
- 查看账户状态应显示为 "Starter" 计划
- 确认获得了新的资源配额

## Starter 计划资源
- **内存**: 512MB
- **CPU**: 共享
- **存储**: 10GB
- **数据库**: PostgreSQL
- **自定义域名**: 支持
- **无限制部署**: 支持

## 费用说明
- **价格**: $5美元/月
- **计费**: 按月自动续费
- **可随时取消**: 在设置中可随时降级或取消
